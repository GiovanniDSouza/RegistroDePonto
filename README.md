# Registro de Ponto

 Log
===================
Um site onde você pode bater o ponto e acompanhar suas horas de trabalho e horas extras ao longo da semana, com a divisão entre horas extras em home office e presenciais.

### Tech used:
- HTML5
- CSS
- JavaScript
